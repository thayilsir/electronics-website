ElectroGlass - Learn Electronics

Welcome to ElectroGlass! This interactive website was created to give students and hobbyists a visually engaging way to explore basic electronics components. Each card below represents a key element in circuits, like resistors, capacitors, and transistors. Click any card to learn how it works, its applications, and practical tips. The goal is to make learning electronics fun, colorful, and modern while remaining informative for aspiring engineers.
